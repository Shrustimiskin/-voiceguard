How It Works

MFCC features + CNN classifier — language-agnostic deepfake detection.

01
Audio Capture
Voice recorded at 16kHz mono or loaded from file. Padded/trimmed to exactly 3 seconds. Silence is detected and rejected before processing.
02
MFCC Extraction
40 Mel-Frequency Cepstral Coefficients extracted using librosa — output is a 40×301 matrix, like a fingerprint of the voice.
03
CNN Analysis
3-layer CNN with BatchNorm and Dropout analyses MFCC patterns. 634,242 trainable parameters. Runs on CPU — no GPU needed.
04
Verdict
Softmax outputs REAL% and FAKE% confidence scores. If fake score exceeds 50%, the audio is flagged as AI-generated.

About This Project

Built to address voice deepfake threats in Indian languages — a largely unaddressed problem.

English
Hindi — हिंदी
Kannada — ಕನ್ನಡ
Marathi — मराठी
Telugu — తెలుగు
Dataset
Training Data
DEEP-VOICE celebrity dataset (English) + Indian Languages Audio Dataset + gTTS synthesised fakes. 412 balanced samples total.
Model
VoiceCNN
3-layer CNN on MFCC features. 634K parameters. 25 epochs on CPU. 95% test accuracy, 92.68% validation accuracy.
Limitation
Known Boundary
Neural voice cloning (ElevenLabs, RVC) is not reliably detected — future work. Other languages beyond the 5 trained may give uncertain results.
